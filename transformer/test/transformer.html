KBPlus Transformer Test

This is a simple form that posts a file and xslt path to the XSLT transformer script on dev.kbplus.

To work it you must have an XML file downloaded from the KBPlus app representing the titles that you want to transform.

Example curl command

curl -X POST -F "xml=@./titles_listing_2210.xml;type=text/xml" -F "path=/home/kbwww/www/stylesheets/serialssolutions.xslt" http://dev.kbplus.ac.uk/cgi-bin/KBPlus_transformer.cgi
