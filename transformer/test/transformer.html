KB+ Transformer Test

This is a simple form that posts a file and xslt path to the XSLT transformer script on the various KB+ servers.

To work it you must have an XML file downloaded from the KB+ app representing the titles that you want to transform.

Post to dev.kbplus

XSLT:
File:

Example curl command

This does the same thing from the CLI:

curl -X POST -F "xml=@./titles_listing_2210.xml;type=text/xml" -F "path=/home/kbwww/git/KBPlus/transformer/stylesheets/serialssolutions.xslt" http://dev.kbplus.ac.uk/cgi-bin/KBPlus_transformer.cgi
